# battletanks-v1 Onboarding Example

1. Add `templates/battletanks-v1/template.json` with immutable/editable contract split.
2. Add rulesets (`default`, `fast-rounds`) and `scenario.schema.json`.
3. Register in `templates/registry.json`.
4. Verify API listing from registry-backed endpoint.
5. Run certification gates and archive output.
6. Ensure local-tank fallback + keyboard capture + auth profile chain safeguards remain in generated runtime.
