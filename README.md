# viverse-sdk-skills

Independent VIVERSE SDK skills repository for LLM-friendly skill discovery and loading.

## Structure
- `skills/`: skill content (SKILL.md, patterns, examples, rules)
- `catalog/`: machine-readable index and routing
- `schemas/`: JSON schemas for validation
- `bundles/`: cross-skill shared guidance
- `scripts/`: validation/index build utilities

## Status
Initial scaffold created from `viverse_ai_agent/skills`.
